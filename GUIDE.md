# gterm Keyboard Shortcuts Guide

This guide provides a complete reference of keyboard shortcuts available in gterm.

## File Operations

| Action | Shortcut |
|--------|----------|
| New file | `Ctrl+N` |
| Open file | `Ctrl+O` |
| Save | `Ctrl+S` |
| Save As | `Ctrl+Shift+S` |
| Save All | `Ctrl+Shift+A` |
| Close file | `Ctrl+W` |
| Close All | `Ctrl+Shift+W` |
| Quit | `Ctrl+Q` |

## Editing

| Action | Shortcut |
|--------|----------|
| Undo | `Ctrl+Z` |
| Redo | `Ctrl+Y` |
| Cut | `Ctrl+X` |
| Copy | `Ctrl+C` |
| Paste | `Ctrl+V` |
| Select All | `Ctrl+A` |
| Delete Line | `Ctrl+K` |
| Duplicate Line | `Ctrl+D` |
| Move Line Up | `Alt+Up` |
| Move Line Down | `Alt+Down` |

## Search & Replace

| Action | Shortcut |
|--------|----------|
| Find | `Ctrl+F` |
| Find Next | `Ctrl+G` or `F3` |
| Find Previous | `Ctrl+Shift+G` or `Shift+F3` |
| Replace | `Ctrl+H` |
| Go to Line | `Ctrl+L` |

## Navigation

| Action | Shortcut |
|--------|----------|
| Go to Matching Brace | `Ctrl+B` |
| Next Tab | `Ctrl+PageDown` |
| Previous Tab | `Ctrl+PageUp` |
| Switch to Tab 1-9 | `Alt+1` through `Alt+9` |
| Switch to Last Tab | `Alt+0` |

## View & Focus

| Action | Shortcut |
|--------|----------|
| Focus Editor | `F2` |
| Focus File Tree | `F3` |
| Focus Terminal | `F4` |
| Toggle Sidebar | `Ctrl+Shift+B` |
| Toggle Terminal | `Ctrl+Shift+T` |
| Zoom In | `Ctrl++` |
| Zoom Out | `Ctrl+-` |
| Reset Zoom | `Ctrl+0` |

## Mouse Actions

| Action | Mouse Event |
|--------|-------------|
| Position cursor | Left click in editor |
| Select text | Left click + drag |
| Extend selection | Shift + left click |
| Select word | Double click |
| Select line | Triple click |
| Open file | Left click on file tree item |
| Expand/collapse directory | Left click on directory |
| Switch tab | Left click on tab |
| Close tab | Middle click on tab or click X |
| Resize panes | Drag pane dividers |
| Scroll | Scroll wheel |
| Context menu | Right click |
| Focus terminal | Click in terminal |
